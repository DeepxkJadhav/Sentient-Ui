# ✨ Overview

**Sentient-UI** is a futuristic multi-agent orchestration platform where autonomous AI agents collaborate, reason, negotiate, and execute workflows in parallel.

Instead of a single AI chatbot, Sentient-UI creates an entire **AI operating system** with:

- 🧠 Shared long-term memory
- 🤖 Specialized autonomous agents
- ⚡ Parallel execution pipelines
- 🌐 Federated agent communication
- 🎯 Goal decomposition engine
- 🔥 Real-time swarm orchestration

> Sentient-UI is not a chatbot.  
> It is infrastructure for collective machine intelligence.

# 🚀 Quick Start

## 1. Clone Repository

```bash
git clone https://github.com/DeepxkJadhav/Sentient-UI.git
cd Sentient-UI
```

---

## 2. Install Dependencies

### Frontend

```bash
cd apps/web
npm install
```

### Backend

```bash
cd apps/api
pip install -r requirements.txt
```

---

## 3. Configure Environment Variables

### Backend `.env`

```env
DATABASE_URL=
REDIS_URL=
OPENAI_API_KEY=
ANTHROPIC_API_KEY=
JWT_SECRET=
QDRANT_URL=
```

### Frontend `.env.local`

```env
NEXT_PUBLIC_API_URL=
NEXT_PUBLIC_WS_URL=
```

---

## 4. Start Development Servers

### Frontend

```bash
npm run dev
```

### Backend

```bash
uvicorn main:app --reload
```
